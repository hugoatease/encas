Bienvenue à la Maison ISEN

Gérez les comptes

L'interface de gestion des comptes vous permet de visionner les détails d'un compte, de l'éditer, de le supprimer et de voir ses transactions associées de manière simple.

Effectuez des transactions

La caisse vous permet de provisionner un compte et d'effectuer des opérations.
En cas d'erreur, il est toujours possible de révoquer une transaction.

Administrez les comptes

Encas offre des fonctions d'administration couvrant les comptes et les transactions et vous permettant d'avoir une vue d'ensemble sur l'instance installée.

Open Source

Encas est un logiciel libre sous licence GNU GPL v3.
Vous pouvez consulter son code sur GitHub et nous faire part de vos suggestions ou de vos rapports de bugs.
